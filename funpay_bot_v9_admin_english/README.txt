FunPay Deals Bot — версия с админ-панелью после ClezzyKryt и RU/EN интерфейсом

Запуск:
1. Установите Python 3.11 или новее.
2. Установите зависимость:
   pip install -r requirements.txt
3. Запустите:
   python funpay_deals_bot_v7_persistent.py

Что изменено:
- После активации ClezzyKryt у пользователя сохраняется доступ к тестовой
  админ-панели и в главном меню появляется отдельная инлайн-кнопка
  «Админ панель».
- Код ClezzyKryt по-прежнему доступен всем пользователям:
  /ClezzyKryt
  /promo ClezzyKryt
- Покупатель после подтверждения перевода больше не получает сообщение
  «Сделка успешно подтверждена / Деньги были отправлены на ваш баланс бота».
  Это сообщение отправляется только продавцу. Покупателю показывается
  подтверждение, что средства отправлены продавцу.
- При первом обычном запуске через /start показывается выбор языка
  с отдельным изображением language_selection.jpg. После выбора язык
  сохраняется, а следующие команды /start сразу открывают главное меню.
- В начальном выборе и в разделе «Язык» используются две инлайн-кнопки:
  «Русский» — premium emoji ID 5449408995691341691;
  «English» — premium emoji ID 5202021044105257611.
- Выбранный язык и факт прохождения начального выбора сохраняются в
  bot_data.json. После выбора английского
  переводятся интерфейсные сообщения, кнопки и всплывающие уведомления.
- Исправлена обработка локализованных сообщений при редактировании экранов,
  FSM-вводе и отправке уведомлений другому участнику сделки.
- В разделе «Создать сделку» добавлен выбор из 10 готовых игр/категорий
  с premium emoji и кнопка «Другое». После нажатия «Другое» пользователь
  вводит собственное название. На каждом этапе сохранена навигация «Назад».

Сохранение данных:
- Реквизиты, балансы, сделки, промокоды, заявки на вывод, доступ ClezzyKryt
  и выбранный язык сохраняются в bot_data.json рядом с файлом бота.
- Не удаляйте bot_data.json при обновлении, если требуется сохранить данные.

Другие общие коды:
- /Work или /promo Work — включает безлимитный тестовый баланс.

Обычные промокоды:
- Создать: /addpromo КОД ВАЛЮТА СУММА [ЛИМИТ]
- Активировать: /promo КОД
- Отключить: /delpromo КОД
- Список: /promos

Команды /addpromo, /delpromo и /promos доступны только ID из ADMIN_IDS.

Изображения разделов:
- Для «Реквизиты», «Создать сделку», «Профиль», F.A.Q, «Вывод средств»,
  «Активные сделки» и «Активные выводы» добавлены отдельные RU/EN-баннеры.
- Баннер автоматически выбирается по сохранённому языку пользователя.
- При переходе между разделами изображение заменяется в том же сообщении.
- В разделе «Язык» используется существующее изображение language_selection.jpg.
